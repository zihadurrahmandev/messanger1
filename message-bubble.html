Message Bubble Component

Reusable component for chat message bubbles with sent and received variants.

Hello! This is a received message.

10:30 AM

This is a sent message.

10:31 AM

Jane Smith

This is a received message with sender indicator.

10:32 AM

Usage Example

<!-- Received Message -->
<div class="flex justify-start">
    <div class="max-w-[70%] bg-white text-[#050505] rounded-tl-lg rounded-tr-lg rounded-bl-lg border border-[#DBDBDB] px-4 py-2 shadow-sm">
        <p>Hello! This is a received message.</p>
        <span class="text-xs text-[#65676B] block mt-1 text-right">10:30 AM</span>
    </div>
</div>

<!-- Sent Message -->
<div class="flex justify-end">
    <div class="max-w-[70%] bg-[#0084FF] text-white rounded-tl-lg rounded-tr-lg rounded-br-lg px-4 py-2 shadow-sm">
        <p>This is a sent message.</p>
        <span class="text-xs text-white/70 block mt-1 text-right">10:31 AM</span>
    </div>
</div>
